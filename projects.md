This page is the public hub for my ongoing software and research projects. I use GitHub for day-to-day task tracking, while this page collects high-level summaries, milestones, code links, and public outputs in one place.

For collaborators, GitHub remains the source of truth for issues, commits, pull requests, and release history.

__How I manage projects__
---

- **Planning:** GitHub Issues and GitHub Projects
- **Development:** repository branches, commits, and release tags
- **Public updates:** this page, the project README, and linked papers or talks

__Active project hubs__

### Multi_Region_CliPP
Fast and accurate multi-sample subclonal reconstruction for cancer evolution studies.\
**Repository:** [yuadamding/Multi_Region_CliPP](https://github.com/yuadamding/Multi_Region_CliPP)\

### Single-shot differentially private federated learning
Privacy-preserving function-on-function learning for telemedicine and distributed healthcare analytics.\
**Repository:** [yuadamding/s2dp_federated_study](https://github.com/yuadamding/s2dp_federated_study)\

Machine learning methods for telemedicine-assisted screening and healthcare decision support.\
**Repository:** [yuadamding/AI_Obstructive_Sleep_Apnea](https://github.com/yuadamding/AI_Obstructive_Sleep_Apnea)\
**Focus:** maintained research code and publication-linked results
